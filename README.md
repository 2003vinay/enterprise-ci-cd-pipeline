# Enterprise CI/CD Pipeline using GitHub Actions, Docker & AWS

## 📌 Project Overview

This project demonstrates a production-style CI/CD pipeline built using GitHub Actions. The pipeline automates code validation, testing, containerization, and deployment to AWS.

The goal is to simulate an enterprise DevOps workflow using modern CI/CD practices.
